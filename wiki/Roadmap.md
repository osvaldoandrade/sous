# Roadmap

This file lists post-v0.1 items.

## Runtimes

- Add `cs-python` runtime with the same capability model.
- Add `cs-wasm` runtime for compiled languages.

## Triggers

- Add cron schedules.
- Add codeQ subscription triggers (event-driven).

## Packaging

- Add dependency bundles for JS with import maps.
- Add signed bundles with tenant keys.

## Observability

- Add per-node decision tracing for agent trees.
- Add activation sampling controls.

## Security

- Add secret provider integration with an external vault.
- Add per-tenant network egress gateways.

## Cadence

- Add DecisionTask support for workflow execution in `code-sous`.
- Add activity result payload codecs per tasklist.
